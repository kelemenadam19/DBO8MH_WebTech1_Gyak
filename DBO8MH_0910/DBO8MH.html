Legfontosabb Git parancsok

Alapvető parancsok

  • git init - Új Git repository létrehozása
  • git clone [url] - Repository másolása
  • git status - Munkakönyvtár állapotának megjelenítése
  • git add [fájl] - Fájl hozzáadása a staging area-hoz
  • git commit -m "üzenet" - Változtatások véglegesítése

Branchek kezelése

  • git branch - Ágak listázása
  • git branch [ág-neve] - Új ág létrehozása
  • git checkout [ág-neve] - Ág váltása
  • git merge [ág-neve] - Ág egyesítése

Remote repository kezelése

  • git remote add [név] [url] - Távoli repository hozzáadása
  • git push [távoli] [ág] - Változtatások feltöltése
  • git pull [távoli] [ág] - Változtatások letöltése
  • git fetch [távoli] - Távoli változtatások letöltése

Változtatások nyomon követése

  • git log - Commit history megjelenítése
  • git diff - Módosítások megjelenítése

Visszavonási parancsok

  • git checkout -- [fájl] - Módosítások elvetése
  • git revert [commit] - Commit visszavonása
